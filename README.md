# smartphone-app

## 📱 アプリ操作概要（使い方）

このアプリは、あなたの利用状況に合わせて大手キャリア・サブブランドの中から一番安いプランを、可愛いジュエリーボックス風のデザインで算出するシミュレーターです。

### ⚙️ 操作手順

| ステップ | 操作項目 | 操作内容 / 選択肢のポイント |
|:---|:---|:---|
| **1** | **データ通信量の設定** | 「💖 1ヶ月に必要なデータ通信量」のスライダーを左右に動かして、必要なギガ数（1GB〜120GB）を指定します。 |
| **2** | **基本情報の入力** | 以下のドロップダウンを選択します。<br>・**契約者の年齢**: 学割（18歳以下）やシニア割（60歳以上）の判定に影響します。<br>・**通話オプション**: 5分かけ放題、24時間かけ放題の有無を選びます。<br>・**家族の回線数**: 家族割（1回線〜3回線以上）の割引額に影響します。<br>・**クレジットカード決済**: 対象のカード決済割引（dカード、au PAYカード、PayPayカード等）を反映します。 |
| **3** | **セット割引の選択** | 適用可能なインフラセット割のチェックボックスをONにします。<br>・🏠 **自宅のネット回線セット割**<br>・⚡ **でんき割** |
| **4** | **シミュレーション実行** | 画面中央にある **「料金を調べる ✨」** ボタンをクリックします。 |

---

### 💎 結果の仕組みと特殊な計算ロジック（仕様）

ボタンを押すと、下部のショーケース（結果エリア）に**「月額料金が安い順（税込）」**に各キャリアのプランが香水瓶（ジュエリーボトル）風のカードで並び替わって表示されます。各ボトルをクリックすると、各社の公式サイトへ直接アクセスできます。

#### ⚠️ UQモバイル（UQトクトクプラン）に関する重要ロジック
本アプリでは、実際のキャリア仕様に準拠した以下の高度な割引判定ロジックを搭載しています。
* **セット割の重複回避**: 
  「自宅のネット回線セット割」と「でんき割」は割引率が同じ（-1,100円）です。**両方にチェックを入れた場合でも、二重に割引（-2,200円）されることはなく、片方のみ（-1,100円）が安全に適用**されます。
* **割引内訳の可視化**: 
  ボトル内の「詳細（detail）」テキストに、現在「自宅セット割」「でんき割」「家族割」のどれが適用されてその料金になっているかがひと目でわかるメッセージが表示されます。
